MySQL-01-数据库与SQL相关概念

数据库与SQL相关概念

  • 引入
  • 为什么要学习数据库
  • 概念
  • 数据库的特点

引入

服务器从数据库中拿到数据。

mysql是互联网公司最常用的。

为什么要学习数据库

内存：容易丢失

文件：不容易查询

数据库的好处

  1. 持久化
  2. 方便查询

概念

DB：数据库，保存了一系列有组织的数据的容器。

DBMS：数据库管理系统，DB是通过DMBS创建和操作的容器。有时候会简称数据库软件。

SQL：Structure Query Language，结构化查询语言。

SQL的优点

  1. 通用
  2. 简单易学
  3. 虽然简单但是强大

数据库的特点

  1. 先讲数据放在表中，表在放到库中。

  2. 一个数据库中可以有多个表，每个表都有一个名字，用来标识自己。表名具有唯一性。

  3. 表具有一些特性，特性定义了数据在表中如何存储，类似java中类的设计。（关系模式）

  4. 表的列称为字段（属性）。所有表都是由一个或多个列组成的，类似于java中的属性。

  5. 表的行称为元组。类似于java中的对象。
